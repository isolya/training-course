2.3 Вводим с консоли

Как сделать программу пригодной для использования?

Мы, программисты, как авторы кода умеем его читать, разбираться в нем и изменять ради своих нужд. Но зачастую программы мы пишем не для программистов, а для людей, которые читать и изменять код не умеют. Чтобы они могли использовать ваши программы у них должен быть какой-то пользовательский интерфейс. Давайте научимся делать самый простой вариант такого интерфейса – консольный.

Поведение у консольного интерфейса обычно следующее – пользователь запускает программу, видит, что ему что-то написала консоль. Вводит в нее какие-то данные и получает результат работы на основании своих данных.

Выводить данные мы уже умеем с помощью print, но как их вводить в программу? Для этого в Python есть функция input.

name = input()
print("Привет", name, "!")

Теперь если пользователь введет свое имя, то он получит результат Привет Иван !. Единственный недостаток такой программы – это то, что она не выводит для человека никаких подсказок. Как понять, что нужно вводить? Подсказку можно передать аргументом в функцию input.

name = input("Введите ваше имя: ")
print("Привет", name, "!")
