5.1 Подготовка

В проекте "Весёлая ферма" мы узнаем новый способ управления игровым персонажем и научимся отслеживать параметры по мониторам переменных


Фон для игры мы не будем брать из библиотеки, а полностью нарисуем сами.

В графическом редакторе выбери инструмент "Прямоугольник" и настрой цвет с параметрами: Цвет 35, Насыщенность 75, Яркость 53. Получится темно-зеленый цвет.



После этого изобрази прямоугольник размером во всю рабочую область экрана.


Научимся рисовать озеро. Для этого выберем подходящий цвет, например с такими настройками:



Для рисования озера воспользуемся инструментом "Круг"



Обрати внимание, что озеро получилось с черным контуром. Этот контур можно удалить, изменив значение в окошке "Контур" на 0. Для этого озеро должно быть сначала выбрано!



Не забывай, что если какое-то действие в графическом редакторе не получается выполнить сразу желаемым образом, то всегда можно воспользоваться кнопкой "Отмена" для отмены любого количества последних действий.
Теперь научимся изменять форму озера таким образом, чтобы оно не было правильной а формы, а было похожим на реальное озеро. Для этого нам поможет инструмент "Изменение формы"



Теперь на нашем озере появились точки привязки, потянув за которые можно изменить форму озера. Кроме того в выбранной точке привязки можно потянуть за дополнительные "усики", еще больше изменяя форму. Для получения более точной формы можно устанавливать дополнительные точки привязки, просто щелкнув мышью на контур фигуры. Щелкнув же на саму точку привязки, ее можно удалить.



Поэкспериментируй с точками привязки и получи желаемую форму озера.


Аналогичным образом нарисуем два камня на фоне. Потренируйся создавать различные формы в редакторе изображений.



Теперь нарисуем забор. Выбери для него подходящий цвет и нарисуй верхнюю горизонтальную доску.



Поскольку нижняя доска точно такая же, как и верхняя, нет необходимости рисовать ее заново. Мы можем воспользоваться инструментом копирования.



После нажатия на кнопку "Копировать" кнопка "Вставить" станет активной.



При нажатии на нее скопированный элемент из буфера обмена будет вставлен рядом с первым. Теперь его можно переместить мышью на нужное место.



Используя инструмент копирования, можно дорисовать остальные доски в заборе.



Главным персонажем нашей игры будет цыпленок, которого мы будем кормить. Спрайт можно найти в библиотеке спрайтов.



Для игры нам понадобится еще один спрайт - крошки, которые будет клевать цыпленок. Их мы нарисуем в графическом редакторе. Для этого необходимо в окне спрайтов выбрать пункт "Нарисовать".



Поскольку крошки должны быть небольшого размера, для удобства их рисования мы можем увеличить масштаб при рисовании, то есть приблизить изображение. Для этого достаточно нажать на увеличительное стекло с плюсиком справа от области рисования. Чтобы уменьшить масштаб, нужно нажать увеличительное стекло с минусом. Чтобы вернуться к исходному масштабу, достаточно нажать значок "равно".



Теперь нарисуем несколько кругов так, чтобы было похоже на крошки хлеба.




Очень важно понимать, что при рисовании любого спрайта нужно знать, где находится центр его костюма. В редакторе изображений центр костюма отображается маленьким кругом с перекрестием.

В данном случае все крошки должны располагаться неподалеку от центра спрайта. Тогда программа будет работать правильно
