2.1 Учим компьютер выводить результат

Как вывести результат работы программы?


Первое чему мы научимся – выводить информацию в нашу консоль. Это нужно, чтобы видеть результаты работы нашей программы.

Для вывода используется функция print. Чтобы вызвать ее достаточно добавить круглые скобки.

print()

Попробуйте запустить программу выше и вы увидите, что у вас печатается пустая строка. А как же выводить текст? Для этого достаточно добавить его внутрь скобок в кавычках.

print('python for backend')

Абсолютно не важно какие кавычки вы используете для этого, можно и так:

print("python for backend")

Если говорить заумными терминами команда print – это функция.

Функция – это такой литерал языка, который позволяет выполнять какие-то готовые действия. Например функция print выводит аргумент, который вы указали в круглых скобках в консоль.

Аргумент – это то, что вы передаете в вашу функцию. В примере выше аргументом является строка "python for backend".

Давайте выведем несколько строк в рамках одной программы. Например, пусть слова python for backend будут каждое на новой строке.

print('python')
print('for')
print('backend')

Функция print каждый раз выводит аргумент на новой строке. Это ее поведение по умолчанию.

Теперь разделим наш вывод пустыми строками, чтобы текст не слипался.

print('python')
print()
print('for')
print()
print('backend')

Продолжаем разбираться с print

В функцию print также можно передать несколько аргументов, если перечислите их через запятую.

# – это комментарий к коду, мы с его помощью будем обозначать что выведет программа, или писать какие-то заметочки с пояснениями

print('python', 'backend') # python backend
print('python', 'for', 'backend') # python for backend

Аргументы, которые вы перечисляете через запятую называются позиционными. Иногда в функциях важно на какой позиции стоит параметр, а иногда нет. В функции print от позиции зависит в какой последовательности будут напечатаны аргументы.

Позиционные аргументы при выводе разделяются пробелами. Разделение аргументов еще называют сепарацией (от слова separate - разделять). Вы можете заменить разделитель любой другой символ или строку, если укажете именованный аргумент sep.

print('python', 'for', 'backend', sep='->') # python->for->backend

Если указать пробел, то print с sep ничем не будет отличаться от обычного.

print('python', 'for', 'backend', sep=' ') # python for backend
print('python', 'for', 'backend') # python for backend

Есть еще один именованный аргумент end. Он отвечает за то, что будет напечатано в конце каждой строки в консоли. По умолчанию там печатается невидимый специальный символ \n, который переносит следующий вывод на новую строку. Именно по этой причине print каждый раз печатает на новой строке. Давайте попробуем переопределить end.

print('python', end='.')
print('for')
print('backend') # python.for
# backend

Видно, что теперь у нас for прилип к python, потому что не было переноса на новую строку. Если его добавить, то все снова будет нормально.

print('python', end='.\n')
print('for')
print('backend') # python.
# for
# backend

Аргументы можно комбинировать друг с другом.

print('python', 'for', sep=',', end='.\n')
print('backend') # python,for.
# backend

Причем если указать sep в строке где только один позиционный аргумент, то это никак не повлияет на вывод .

print('backend', sep='\n!\n') # backend

Какие еще есть специальные символы, кроме \n?

Специальные символы можно использовать не только в именнованных аргументах, но и везде где есть строки. В примере ниже мы вставляем \n в места, где хотим сделать новую строку.

print('python\nfor\nbackend')

# python
# for
# backend

Из полезных специальных символов есть еще \t. Он позволяет вставить табуляцию в строку.

print('python\tfor\tbackend') # python for backend

Также \ может быть полезен в случае, если вы хотите написать текст с кавычками. Без этого символа программа выведет ошибку.

print('python \'for\" backend') # python 'for" backend

Код который выкидывает ошибку:

print('python 'for" backend')

Такие специальные символы еще называют экранированными последовательностями. Их много и они в основном используются для форматирования текста. Ниже неполная табличка этих символов.
