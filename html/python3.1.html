3.1 Целые числа

Целые числа

Переходим к изучению новых объектов языка Python – целых чисел.

Целые числа – это множество Z из школьного курса математики – все натуральные числа, ноль и обратные им отрицательные числа. Или иными словами целыми числами являются: 0, 5, -5, 345, 45544, -5545 и так далее.

В Python можно записывать целые числа разными способами:

n1 = 123
n2 = +123
n3 = 0
n4 = -123
n5 = 1_000_000 # 1000000
n6 = 1_2_3 # 123

Причем нельзя при записи использовать следующие варианты:

# так нельзя
n1 = 01
n2 = 1,000,000
С целыми числами можно выполнять обычные арифметические операции с помощью Python.

Оператор

Определение

Пример

+

Сложение

print(1 + 12) # 13

-

Вычитание

print(1 - 5) # -4

*

Умножение

print(2 * 5) # 10

/

Деление с плавающей точкой

print(3 / 4) # 0.75

//

Целочисленное деление

print(7 // 4) # 1

%

Остаток от деления

print(7 % 4) # 3

**

Возведение в степень

print(3 ** 5) # 243

Давайте посчитаем с помощью Python несколько примеров.

(100+400−600)∗2

print((100 + 400 - 600) * 2) # -200

53+ 2 2+ 1000

print(5 ** 3 + 2 ** 2 + 100 ** 0) # 130

Как ввести целые числа в программу с клавиатуры?


Целые числа имеют тип int. Объекты этого типа занимают в памяти количество байт соизмеримое размерам числа. Например число 0 занимает 24 байта, а 1 000 000 уже 28 байт.

print(type(1)) # class 'int'

Мы уже знаем, что функция input позволяет ввести строковые значения в программу. Это значит, что арифметика работать не будет.

a = input() # 4
b = input() # 5
print(a + b) # 45

Чтобы интерпретатор воспринимал числа как числа, а не как строки их нужно явно привести к типу int. Это можно сделать так:

a = input() # "4"
b = input() # "5"
a_int = int(a) # 4
b_int = int(b) # 5
print(a_int + b_int) # 9

Python позволяет сокращать размер программ и передавать в качестве аргумента сразу вызов функции, так что программу выше можно сократить.

a = int(input()) # 4
b = int(input()) # 5
print(a + b) # 9

И даже это не предел:

print(int(input()) + int(input())) # 9

Операции деления

Разберем подробнее операции деления. Деление / – это деление с плавающей точкой или, если терминами математики, десятичное деление.

41:5

print(41 / 5) # 8.2

В результате такого деления получается число с плавающей точкой (floating point number). Отличается оно от целого числа наличием точки. Причем, даже если по правилам математики деление дает в остатке 0, то результатом все равно будет являться не целое число.

print(40 / 5) # 8.0

Если в вашем примере есть корни, то они тоже будут давать дробное число в результате.

√169 + 35

print(169 ** (1/2) + 3 ** 5) # 256.0

В примере выше мы возводим в степень 1/2, потому что по правилам математики

Если вас не интересует дробная часть и вы хотите вычислить только целую, тогда следует использовать //.

print(41 // 5) # 8
print(40 // 5) # 8
print(39 // 5) # 7

В результате таких вычислений всегда получается целое число. По сути целочисленное деление всегда округляет в меньшую сторону.

print(-35 // 5) # -7
print(-36 // 5) # -8
print(36 // -5) # -8
print(-40 // 5) # -8
print(-41 // 5) # -9

Если вам нужен остаток от деления, то используйте %.

print(43 % 5) # 3
print(40 % 5) # 0

При таком делении в результате может получиться только число от 0 до n-1. Выходит, что для 5 результатом может быть 0,1,2,3,4.

На первый взгляд этот оператор может показаться бесполезным, но это совсем не так!

К примеру с его помощью можно легко узнать является ли число четным, ведь у четного числа остаток при делении на 2 всегда равен 0.

print(10 % 2) # 0
print(13 % 2) # 1

Или если у вас есть ограниченный набор чисел, которые расположены в виде кольца, яркий пример – это настенные часы.



Как определить куда будет указывать часовая стрелка, если прошло 13 часов? Это можно легко определить для любого количества часов, если сделать замыкание чисел по модулю 12.

print(1 % 12) # 1
print(13 % 12) # 1

Все это применимо к любым кольцевым структурам – стадионам, автомагистралям и так далее.

Деление с остатком в котором участвуют отрицательные числа может показаться странным.

print(-6 % -5) # -1
print(-6 % 5) # 4
print(6 % -5) # -4

Вспоминаем про то, что при делении с остатком может получиться только число, которое находится в кольце от 0 до n-1.

Если посмотреть на пример -6 % 5, то становится понятно, что очевидного ответа -1 в кольце нет, поэтому этот ответ не правильный.

Посчитать остаток от деления можно с помощью несложной формулы: a % b == a - (a // b) * b.

Если вы попробуете разделить на 0 любым из делений, то интерпретатор выкинет ошибку ZeroDivizionError.

print(1 % 0)

Синтаксический сахар

В Python можно делать сокращения кода, например выражения типа a = a + 4 можно заменить на a += 4. Такие сокращения называются синтаксическим сахаром, и с ними код становится проще и выразительнее.

Такой трюк можно проделывать и с остальными операторами.

a = 2
a -= 1
a *= 5
a /= 6
a //= 2
a %= 4
a **= 2

Приоритет операций

При выполнении арифметических вычислений следует учитывать приоритет выполнения каждого оператора. В таблице ниже сверху-вниз идут операторы по уменьшению их приоритета в выражении.

Оператор

Описание

(x)

Скобки

**

Возведение в степень

-x, +x

Унарные операторы

*, /, //, %

Умножение, деление

+, -

Плюс, минус

Рассмотрим код:

x = 5
print(-x**2 + (5+x)) # -15

При расчете арифметического примера сначала выполнится выражение в скобках (5+x), затем x ** 2, после этого к нему добавится -, и только потом две части сложатся.
