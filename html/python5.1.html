5.1 Списки и кортежи

Составные структуры данных. Кортежи.

В предыдущей главе в основном мы работали с простыми типами данных - числами, строками и булевыми значениями. Простыми они являются потому, что внутри нет других типов данных. Теперь пришло время познакомиться со сложными типами данных - они могут содержать в себе как простые, так и сложные типы.

Кортеж - это неизменяемая упорядоченная последовательность элементов. На то, что перед вами кортеж указывает присутствие запятой.

# примеры кортежей

# кортеж из одного элемента
a = 1, # (1,)
a1 = (1,) # (1,)

# кортеж из двух элементов
b = 1,2 # (1,2)
b1 = (1,2) # (1,2)


Если элементы перечислены через запятую без скобок, то Python сам преобразует их в кортеж.

cartoons = 'Rick and Morty', 'The Simpsons', 'Futurama' # ('Rick and Morty', 'The Simpsons', 'Futurama')
# кортеж, содержащий внутри другие кортежи в качестве элементов
status_codes = ((200, 'OK'), (400, 'Bad Request'), (500, 'Internal Server Error'))

# кортеж, составленный из двух предыдущих кортежей
tuples = (cartoons, status_codes) # (('Rick and Morty', 'The Simpsons', 'Futurama'), ((200, 'OK'), (400, 'Bad Request'), (500, 'Internal Server Error')))


В кортеже элементы, так же как и в строках, нумеруются с 0, поэтому удобно обращаться к ним по индексам, дописав к названию кортежа индекс в квадратных скобках. Также можно использовать отрицательные индексы, чтобы получить элементы с конца, т.е. индекс -1 означает последний элемент кортежа.

week = ('Пн', 'Вт', 'Ср', 'Чт', 'Пт', 'Сб', 'Вс')
print(week[2]) # 'Ср'
print(week[6]) # 'Вс'
print(week[-1]) # 'Вс'
print(week[-3]) # 'Пт'


Чтобы узнать длину кортежа, можно использовать функцию len().

week = ('Пн', 'Вт', 'Ср', 'Чт', 'Пт', 'Сб', 'Вс')
print(len(week)) # 7


Срезы в кортежах и списках работают точно так же как в строках.

week = ('Пн', 'Вт', 'Ср', 'Чт', 'Пт', 'Сб', 'Вс')
print(week[2:4]) # ('Ср', 'Чт')

Работа с кортежем


Кортежи встречаются в работе программиста повсеместно, ведь, как мы помним, достаточно перечислить пару элементов через запятую, и Python уже будет считать это кортежем.

Множественное присваивание


При работе с кортежем часто бывает удобнее использовать его элементы как отдельные переменные, чем обращаться к ним каждый раз по индексам. Для этого можно использовать распаковку кортежа.

man = ("Michael", 23, "Programmer")
name, age, profession = man
print(name) # 'Michael'
print(age) # 23
print(profession) # 'Programmer'


Когда происходит присваивание кортежа man другому кортежу (name, age, profession), соответствующим переменным присваиваются соответствующие значения из man.

Распаковку можно так же делать при помощи *.

a, *b = 1,2,3,4,5,6
print(a) # 1
print(b) # [2, 3, 4, 5, 6]

# не нужные значения можно отсечь так
c,d,e, *_ = 1,2,3,4,5,6,7
print(c,d,e) # 1 2 3


Работа с циклом


В уроке с циклами уже встречались кортежи, там поэлементно перебирались их значения. Разберем более сложные случаи использования.

parents = (('Nick Smith', 1980, ('Alice', 'Thomas',),),
('Julia Clark', 1976, ('Oliver', 'George',),),
('Michael Adams', 1993, ('Harry', 'Mia', 'Penny',),),)

for parent in parents:
name, age, children = parent
print(f'Parent: {name}, {age}, children: ', end='')
for child in children:
print(child, end=', ')
print()

# Parent: Nick Smith, 1980, children: Alice, Thomas,
# Parent: Julia Clark, 1976, children: Oliver, George,
# Parent: Michael Adams, 1993, children: Harry, Mia, Penny,

Здесь мы выводим данные о родителях и их детях. В цикле перебираются элементы кортежа, распаковывается каждый кортеж, выводятся данные о родителе, а затем в цикле выводятся имена детей.

Изменение кортежа


Неизменяемость - одно из главных свойств кортежа, поэтому если кортеж уже создан, изменять элементы в нем нельзя, получишь ошибку TypeError. Однако, можно создать новый, добавив в него элементы из старого, с помощью оператора распаковки, о котором мы поговорим немного позже.

Списки


Теперь перейдем к спискам. Список - это тот же кортеж, только изменяемый, который также может хранить элементы самых разных типов. Элементы перечисляются в квадратных скобках [], нумеруется с 0. Список поддерживает все операции, имеющиеся у кортежа.

Попробуем изменить значения списка. Для этого нужно обратиться к элементу списка по индексу и присвоить ему новое значение.

rainbow = ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet']
rainbow[0] = 'Pink'
rainbow[4] = 'Cyan'
print(rainbow) # ['Pink', 'Orange', 'Yellow', 'Green', 'Cyan', 'Indigo', 'Violet']


Элементы списка успешно изменены.

Для создания пустого списка вместо [] можно использовать функцию list(), также с помощью этой функции можно преобразовать кортеж в список.

empty_list = list()
print(empty_list) # []

rainbow = ('Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet')
new_rainbow = list(rainbow)
print(new_rainbow) # ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet']


Бывают ситуации, когда нужно скопировать список в другую переменную и выполнить с ним какие-нибудь операции. Первое, что может прийти в голову, это присвоить переменную с исходным списком другой переменной.

numbers = [1, 3, 4, 6, 7, 8]
other_numbers = numbers
other_numbers[0] = 'word'
print(numbers) # ['word', 3, 4, 6, 7, 8]
print(other_numbers) # ['word', 3, 4, 6, 7, 8]


Однако, это не поможет, значения будут изменяться и в исходном списке. Это происходит из-за того, что в Python переменные не хранят значения напрямую, они хранят ссылки на объекты в памяти. Поэтому когда other_numbers присваивается переменная numbers, в ней сохраняется не новый список, а только ссылка на исходный, и при изменении данных в other_numbers, данные меняются и в исходном списке.

Чтобы скопировать данные списка, также могут пригодиться функция copy и срезы. Такой способ называется поверхностным копированием (shallow copy), и он полезен когда список состоит из простых структур данных.

numbers = [1, 3, 4, 6, 7, 8]

new_numbers_1 = numbers.copy()
new_numbers_2 = numbers[:]

new_numbers_1[0] = 'word'
new_numbers_2[0] = 'he'

print(numbers) # [1, 3, 4, 6, 7, 8]
print(new_numbers_1) # ['word', 3, 4, 6, 7, 8]
print(new_numbers_2) # ['he', 3, 4, 6, 7, 8]


Имейте в виду, если вы храните в списке изменяемые типы данных (сложные структуры), такие как список, множество или другие, то поверхностное копирование не скопирует объекты находящиеся внутри списка. Поэтому для такого копирования нужно использовать deepcopy из модуля copy.

import copy

numbers = [1, [2,3,4], 4, 6, 7, 8]

new_numbers = copy.deepcopy(numbers)

numbers[1][0] = 19
print(new_numbers) # [1, [2, 3, 4], 4, 6, 7, 8]

Методы и функции для работы со списками

Методы

Любой созданный список принадлежит к классу list, в котором определены различные методы для удобной работы с его элементами. Рассмотрим наиболее полезные.

Название

Примечание

Действие

list.append(item)

Добавляет элемент item в конец списка.

list.insert(index, item)

Добавляет элемент item в список по индексу index

list.remove(item)

Удаляет только первое вхождение элемента. Если элемент не найден, вызывает исключение ValueError.

Удаляет элемент item.

list.clear()

Удаляет все элементы списка.

list.index(item)

Если элемент не найден, вызывает исключение ValueError.

Возвращает индекс элемента item.

list.pop([index])

Если индекс не передан, то удаляет и возвращает последний элемент.

Удаляет и возвращает элемент по индексу index.

list.count(item)

Возвращает количество вхождений элемента item в список.

list.sort([key])

По умолчанию сортирует по возрастанию. С помощью key можно передать функцию сортировки.

list.reverse()

Можно заметить, что методы remove() и index() вызывают исключение, если заданного элемента нет в списке. Поэтому перед их использованием важно убедиться в наличии элемента. Делается это с помощью ключевого слова in - он проверяет, есть ли указанный элемент в кортеже, списке, или другой коллекции данных.

months = ['december', 'may', 'june', 'september']
months.index('october')

# ValueError: 'october' is not in list

months = ['december', 'may', 'june', 'september']
if 'october' in months:
months.remove('october')
print(months)

# ['december', 'may', 'june', 'september']

months = ['december', 'may', 'june', 'september']
if 'may' in months:
print(months.index('may'))
months.remove('may')
print(months)

# 1
# ['december', 'june', 'september']

Генераторы списков aka list comprehensions


Наиболее быстро создать список с помощью генератора списка. Это выражение, заключенное в скобки, заключающее в себе такое же выражение, как обычный цикл for.

Сравним реализации решения одной и той же задачи. Код делает одно и то же – переводит в верхний регистр все слова, состоящие из 5-ти букв. Какую из них проще прочитать?

words = ['banana', 'apple', 'orange', 'mango']
upper_words = []
for word in words:
if len(word) == 5:
upper_words.append(word.upper())
print(upper_words) # ['APPLE', 'MANGO']


Перевод в верхний регистр выполняет метод upper(). В остальном этот код должен быть уже понятен.

words = ['banana', 'apple', 'orange', 'mango']
upper_words = [word.upper() for word in words if len(word) == 5]
print(upper_words) # ['APPLE', 'MANGO']


А здесь то же самое делает генератор списка. Наверняка, вы тоже заметили, что вариант с генератором выглядит проще и лаконичнее.

Синтаксис генератора такой (условие указывать необязательно):

[ <действие с элементом> for <элемент> in <список> if <условие, при котором элемент добавляется в список>]

Конечно, генератор применим не для всех задач. Цикл for удобен для выполнения сложных задач, состоящих из нескольких действий над элементами. Генераторы подходят для задач, где в результате требуется список, а к элементам нужно применить простые функции. Также нужно следить за читаемостью кода, например, заменять генератор на цикл for, если генератор занимает несколько строк и сложно понять, что он делает.

Ввод/вывод списков и кортежей
Существует несколько способов ввода и вывода списков, которые могут пригодиться для решения различных задач.

Ввод списка через цикл


Посмотрите на этот листинг кода:

# Реализация с for
new_list = []
n = int(input())
for i in range(n):
new_list.append(int(input()))

# Реализация с генератором списка
new_list = [int(input()) for i in range(int(input()))]


Здесь создается пустой список new_list, считывается количество элементов, а затем n элементов, которые добавляются в список с помощью метода append(). Функция range() создает последовательность чисел от 0 до n.

В данной реализации нужно вводить каждый элементы списка в отдельной строке, т.к. функция int(input()) будет пытаться преобразовать все содержимое строки в целое число и вызовет ошибку, если там будет указано что-то еще, кроме цифр. Для вывода списка в консоль, достаточно передать его в качестве аргумента в функцию print().

Ввод списка в строку


А что, если нам нужно указать все элементы в одной строке? Рассмотрим пример.

nums = [int(i) for i in input().split(' ')] # '1, 2, 3'
print(nums) # [1, 2, 3]


Тут нам на помощь приходит метод split(). Он позволяет разбить строку, полученную от input() на список элементов с указанным разделителем, например, пробелом или запятой. Если ничего не указано, то будет разбиваться по пробелам.

Противоположным к методу .split() можно назвать метод .join() – он склеивает элементы списка в строку, разделяя их указанным сепаратором. Напишем программу для удаления из пути папки "Desktop".

# C:/Users/Masha/Desktop/python_books/CleanCode.pdf
path_list = [i for i in input().split('/') if i != 'Desktop']
# ['C:', 'Users', 'Masha', 'python_books', 'CleanCode.pdf']
new_path = '/'.join(path_list)
print(new_path) # C:/Users/Masha/python_books/CleanCode.pdf

Конструкция match/case с использованием кортежа


В предыдущей главе уже разбиралась конструкция match/case, рассмотрим ее в контексте текущей главы.

commands = ['git commit -m "Fixed"',
'git commit',
'git add /src',
'docker-compose up']

for command in commands:
match command.split():
case 'git', 'add', path:
print(f'В индекс добавлены файлы по пути: {path}.')
case 'git', 'commit':
print('Файлы закоммичены.')
case 'git', 'commit', '-m', message:
print(f'Файлы закоммичены с сообщением {message}')
case 'git', 'push', repository, branch:
print(f'Файлы запушены в {repository} в ветку {branch}.')
case _:
print('Неизвестная команда.')

# Файлы закоммичены с сообщением "Fixed"
# Файлы закоммичены.
# В индекс добавлены файлы по пути: /src.
# Неизвестная команда.


В качестве аргумента может быть передан кортеж или список, а сопоставление аргументов происходит поэлементно.


array


Когда стоит задача обработать большие массивы данных, может пригодиться библиотека NumPy, а конкретнее, определенный в ней array. Это объект, похожий на список, но отличающийся от него тем, что может хранить только один тип данных. Вычислительные операции на большом объеме информации с array выполняются в разы эффективнее, чем со списками.

Создание


Для создания массива array, можно использовать функцию np.array(), который принимает список и тип данных, а возвращает объект типа ndarray.

a = np.array([1, 2, 3], float)
print(a) # [1. 2. 3.]
print(type(a)) # <class 'numpy.ndarray'>


array поддерживает работу с многомерными массивами, а также имеет множество полезных методов, с которыми мы познакомимся позже.
