5.4 Модули и библиотеки

itertools

В ходе нашего курса вы уже успели познакомиться с некоторыми стандартными библиотеками, такими как math и collections. Однако, некоторые очень полезные стандартные библиотеки остались без внимания, и в этом модуле мы постараемся рассмотреть те, которые довольно полезны в Backend разработке, и itertools как раз одна из них.

Конечно, в Python очень много разных библиотек помимо стандартной, этим язык и славится. Чем больше вы будете изучать Python разработку, тем глубже будут ваши знания о тех, уже готовых решениях, которые применимы под решаемые задачи.

Некоторые возможности библиотеки представлены ниже. За полным списком как обычно обращайтесь в гугл.

from itertools import cycle, chain, combinations

for idx, value in enumerate(cycle('abcd')): # создать бесконечный итератор по буквам
print(value)
if idx > 10:
break
# a
# b
# c
# d
# a
# b
# c
# d
# a
# b
# c
# d

a = ['a','b']
b = "abcdfe"
r = [*chain(a, b, range(5))] # объединяет разные итерируемые объекты в один
print(r) # ['a', 'b', 'a', 'b', 'c', 'd', 'f', 'e', 0, 1, 2, 3, 4]

a = combinations('abc', 2) # те самые сочетания из теории вероятностей
print(*a) # ('a', 'b') ('a', 'c') ('b', 'c')


Работа с датой и временем


Сталкиваться с необходимостью использовать дату и время в работе приходится постоянно. Это могут быть либо какие-то банальные дни рождения, время последнего логина пользователя, время создания записи в базе данных и так далее.

Начнем с модуля time. Этот модуль позволяет работать с временем.

import time

seconds = time.time()
print(seconds) # ваше время в секундах с начала эпохи компьютеров


Таким образом можно довольно просто измерить время работы программы.

import time

start = time.time()

res = 0
for i in range(1_000_000):
res += i

print(time.time()-start) # время работы программы


Следующий модуль – datetime. Он позволяет работать с датой и временем, представляя их в понятном для человека формате UTC.

from datetime import datetime, date

d = datetime(year=2022, month=12, day=31, hour=12, minute=12, second=30)
print(d) # 2022-12-31 12:12:30

print(d.date()) # 2022-12-31

print(date.today()) # сегодняшняя дата

d1 = datetime(year=2022, month=12, day=31, hour=12, minute=12, second=30)
d2 = datetime(year=2023, month=7, day=24, hour=16, minute=16, second=22)
print(d2 - d1) # 205 days, 4:03:52


Модуль calendar нужен для работы с календарем и днями недели. Он позволяет определить в зависимости от даты день недели и другие полезные вещи.

import calendar
calendar.weekday(2022, 5, 9) # 0


Оставшийся модуль zoneinfo – это база часовых поясов, которая позволяет работать с часовыми поясами.

from zoneinfo import ZoneInfo
from datetime import datetime

dt = datetime(2022, 11, 13, 12, tzinfo=ZoneInfo("Europe/Moscow"))
print(dt) # 2022-11-13 12:00:00+03:00


Что следует запомнить при работе с датой и временем?


  • Во-первых, вы могли заметить что каждый из этих модулей возвращает время в разных единицах измерения, и зачастую единицы измерения в переменной очень сложно отследить, поэтому называйте свои переменные так: time_secs, time_mins, time_days и т.д.

  • Никогда не пробуйте писать логику работы даты и времени без использования этих библиотек. Если вы вдруг встретили тестовое задание, в котором требуется работа с датами и временем – смело пользуйтесь этими библиотеками. Очень хорошая практика в программировании – писать как можно меньше своего кода.

bisect


Этот модуль очень полезен, если нужно часто вставлять элементы в отсортированный список. При этом не требуется постоянно сортировать массив после вставки. Производительность вставки – O(n).

Плюсом так же является тот факт, что поиск элементов осуществляется за логарифмическое время (алгоритм бинарного поиска).

import bisect

a = [1, 3, 4, 4, 4, 6, 7]
# Найти место для вставки справа
print(bisect.bisect(a, 4)) # 5

# Найти место для вставки слева
print(bisect.bisect_left(a, 4)) # 2

# вставить элемент
bisect.insort(a,5)
print(a) # [1, 3, 4, 4, 4, 5, 6, 7]
