1.2 Установка Python и IDE

Установка интерпретатора

Для начала нам нужно установить интерпретатор языка Python, чтобы мы могли запускать наши программки. Это можно сделать по ссылке и найти интерпретатор который подходит под вашу операционную систему. Обычно достаточно просто нажать на большую желтую кнопку.



Версия языка должна быть выше 3.10.0, чтобы решать задачи на нашем курсе. Теперь дело за вами – загрузите интерпретатор и установите его на свой компьютер.

Если у вас MacOS то вам скорее нужно еще установить XСode. В любом случае если возникнет какая-то проблема при установке – смело копируйте ошибку и вставляйте ее в поисковую строку Google или Яндекса. Вам подскажут, что нужно установить дополнительно.

После установки откройте свой терминал или консоль. В MacOS это маленькое черное окошко. И введите туда python и первые две цифры вашей версии, например для 3.11 нужно ввести python3.11.



Установка IDE

Мы рекомендуем вам использовать PyCharm. Его можно скачать по ссылке. Используйте Community версию, она бесплатная.



Когда установите IDE и запустите его у вас появится окошко где нужно нажать кнопку New Project.



Тут нужно будет выбрать кнопку создания нового окружения. Оно изолирует ваш интерпретатор проекта от главного интерпретатора. Это полезно для тех случаев, когда вы не хотите засорять память компьютера лишними библиотеками – все будет храниться в папке проекта.



После создания проекта нажмите правой кнопкой мыши на серую папку и создайте файл Python и назовите его main.



В нем вы будете писать свою первую программу. Запустить ее можно либо по зеленой кнопке, либо через терминал, если ввести в нем python main.py.
