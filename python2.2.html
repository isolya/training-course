2.2 Типы данных и переменныет

В Python все является объектом

Все данные в компьютере представляют из себя последовательность битов. Представить ваш жесткий диск размером в терабайт памяти можно в виде огромного пространства, которое чем-то заполнено. Программа, написанная на Python с помощью операционной системы имеет доступ к жесткой и оперативной памяти и может ее использовать для выполнения расчетов или сохранения результатов своей работы.

Давайте посмотрим на обычную программу, которая выводит строку на экран.

print("python")

Когда вы запускаете программу, то в ней сначала создается объект строки "python", а затем объект print. Оба эти объекта занимают место в оперативной памяти. Конкретно объект строки "python" занимает 55 байт, а объект print сам по себе весит 72 байта, а со скобками 16 байт, так как возвращает другой объект None.

На количество памяти, которое занимает объект влияет множество факторов, первый из которых – его тип. Давайте разберемся какие типы данных есть в Python!

Типы данных

Типы в Python делятся на две большие группы: изменяемые и неизменяемые. Изменяемость означает, что вы можете взять объект, который положили в память и поменять что-то в его внутренностях. Обычно у неизменяемых объектов есть множество преимуществ с точки зрения производительности. Например они занимают меньше памяти.

Когда вы работали с print вы использовали строковые параметры. Строка – это один из неизменяемых типов Python и она имеет тип str. Чтобы узнать тип вашего объекта можно использовать функцию type.

print(type('python')) # '<'class 'str'''>

Мы будем знакомиться с разными типами данных Python по мере прохождения курса, а пока давайте узнаем что такое переменная.

Переменные

Часто бывает полезно сохранить значение объекта в памяти, чтобы с ним было удобно работать в программе. Например есть у нас строка с именем: "Иван", и вы хотите вывести ее 5 раз в консоль. Вот такая у вас необычная задача. Код может выглядеть так:

print("Иван")
print("Иван")
print("Иван")
print("Иван")
print("Иван")

Через какое-то время вам разонравилась эта строка и вы решили заменить ее на другую. Для этого нужно целых 5 изменений! Это очень не удобно и было бы просто прекрасно, если вы сохранили ваше значение в памяти и потом обращались к нему по определенному имени. Такие имена называются переменными.

name = "Геральт"
print(name)
print(name)
print(name)
print(name)
print(name)

Переменная – имя для значения в памяти вашего компьютера. Для них есть некоторые правила:

  • Они могут содержать только следующие символы:

    • буквы в нижнем и верхнем регистре;

    • цифры

    • нижнее подчеркивание

  • Они чувствительны к регистру – это означает, что name и Name две разных переменных.

  • Они не могут начинаться с цифры

  • Они не могут совпадать с зарезервированными словами, такими как False, None, True, and и другими

Рассмотрим подробнее нашу переменную name. Между именем переменной и ее значением стоит знак =. В Python это называют присваиванием.

name= "Геральт"

Присваивание – это когда вы создаете объект в памяти и связываете его с именем, по которому будете к нему обращаться.



Переменная ссылается на ячейку в памяти компьютера после присваивания. В случае если вы захотите изменить значение переменной на другое, то вы создадите новый объект в памяти и сделаете на него ссылку.

Что же произойдет со объектом "Геральт"? Он что, будет просто кушать память моего компьютера? Некоторое время да. Потом сборщик мусора увидит, что на этот объект нет ссылок, уничтожит его и тогда память освободится.

Теперь, когда мы знаем о переменных, мы можем писать очень удобный в использовании код.

name = "Геральт"
age = "100"
print(name, age) # Геральт 100

Копирование переменных

Как создать две переменные со значением "hello"? Есть несколько способов:

Первый – тот, что мы уже знаем.

s1 = "hello"
s2 = "hello"

Второй – более элегантный. Тут по цепочке присваивается значение сначала в s2, а затем в s1.

s1 = s2 = "hello"

А как можно поменять местами значения двух переменных? Если вы пришли из другого языка, то скорее всего могли подумать о таком способе:

s1 = "first"
s2 = "second"
temp = s1 # first
s1 = s2 # second
s2 = temp # first

Однако в Python можно написать это проще с помощью множественного присваивания.

s1 = "first"
s2 = "second"
s1, s2 = s2, s1

Как выбрать хорошее имя для переменной?

Очень важно правильно именовать ваши переменные. Хорошее название переменной должно отражать ее содержимое. Есть несколько простых правил по именованию:

  • Никогда не используйте транслит – только английский язык.

  • Используйте короткие имена по типу s, a, n только для переменных местного назначения. Для задач на степике или маленькой функции в самый раз.

  • Если вы хотите назвать переменную двумя и более словами пишите через нижние подчеркивания: my_variable.
